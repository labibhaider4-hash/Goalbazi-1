# Goalbazi Website

This folder now contains a working local web app with:

- a Python backend
- a SQLite database
- a custom embedded SVG logo
- API-backed profile, games, chat, attendance, turf bookings, and league data

## Run locally

```bash
cd website
python server.py
```

Then open:

```text
http://127.0.0.1:8000
```

The SQLite database will be created automatically in:

```text
website/data/goalbazi.db
```
